Cyclophiops is a lightweight computer system status analysis tool designed for privacy awareness and educational purposes related to computer systems.

## Requirements

- Windows 7 or later
- .NET Framework 4.7.2 or higher
- Administrator privileges (required for certain features)

---

## Notes

- Some features require administrator privileges
- All collected information is **read-only** and does not modify system configuration
- This software **does not include** background services, headless execution modes, or any network-based data upload functionality
